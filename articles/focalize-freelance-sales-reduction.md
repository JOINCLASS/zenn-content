## はじめに: フリーランスの「営業あるある」、心当たりありませんか？

フリーランスや個人事業主として独立すると、本業のスキル以外に「営業」という仕事が丸ごとのしかかってきます。

会社員時代は営業部門がやってくれていたこと、全部自分でやらないといけない。

- 問い合わせメールが来たけど、返信が3日後になってしまった
- 「あの案件、いま見積もり出したっけ？」と記憶が曖昧になる
- Excelの案件管理表、先月から更新していない
- 名刺交換したあの人にフォローメールを送ろうと思って、もう2週間経っている

これ、全部わたし自身の経験です。

フリーランスにとって営業事務は「やらないと売上が止まる」のに「時間を取られすぎると本業ができない」という厄介な存在です。本記事では、この営業事務の大部分をAIに任せて、本業と商談に集中できる環境をつくる方法を紹介します。

## 営業事務の80%は「転記作業」だった

まず、自分の営業事務にどれだけ時間を使っているか、内訳を整理してみましょう。

フリーランスの営業事務を分解すると、おおよそこんな配分になります。

| 作業内容 | 月あたり目安 | 割合 |
|---------|------------|------|
| メールの確認・返信 | 4時間 | 25% |
| 案件情報の転記（メール→Excel/CRM） | 4時間 | 25% |
| 日程調整のやりとり | 3時間 | 19% |
| ステータス管理（進捗更新） | 2時間 | 13% |
| フォローアップの確認・送信 | 2時間 | 13% |
| 提案書・見積書の作成 | 1時間 | 5% |

上の表をよく見てください。**転記・入力・ステータス更新で全体の約60%を占めています**。さらに日程調整まで含めれば約80%です。

これらはどれも「判断」が必要な仕事ではありません。メールに書いてある情報を別の場所に書き写す、カレンダーの空きを確認して候補日を提示する、といった**定型的な作業**です。

つまり、営業事務の80%は「人間がやる必要のない作業」なのです。

## AI営業自動化で何が変わるか

では、AIに転記作業を任せると具体的にどう変わるのか。Before/Afterで見てみましょう。

### Before: すべて手作業の場合

```
1. 問い合わせメールを受信（Gmail）
2. メールを読んで、会社名・担当者名・連絡先をExcelに手入力
3. 相手の希望日時を確認し、自分のカレンダーと照合
4. 候補日を返信メールで送る
5. 日程が決まったらカレンダーに手動登録
6. Excel上のステータスを「商談調整中」に更新
→ ここまでで約15〜20分。月10件なら3時間以上。
```

### After: AIに転記を任せた場合

```
1. 問い合わせメールをFocalizeに転送（1クリック）
2. AIが会社名・担当者名・連絡先・希望日時を自動抽出
3. カンバンボードに案件として自動登録
4. Googleカレンダーと連携して日程候補を自動提示
→ 所要時間：転送の10秒だけ。月10件でも2分未満。
```

**15〜20分の作業が10秒になる**。これが「80%削減」の中身です。

大事なのは、空いた時間で提案内容を練ったり、既存クライアントとの関係構築に使えること。売上に直結する「本当の営業」に集中できるようになります。

## 月10件までなら無料で始められる

「とはいえ、ツール導入にお金がかかるんでしょう？」

これは当然の疑問です。フリーランスにとって月額の固定費は慎重に判断したいところ。

わたしが実際に使っている [Focalize](https://focalize.jp) は、**月10件まで無料**で使えます。

フリーランスの案件数を考えると、月10件で足りる人も多いのではないでしょうか。まずは無料プランで試してみて、案件が増えてきたらProプラン（月額2,980円/ユーザー、100件まで対応）に移行する、という段階的な導入ができます。

ちなみに、CRMを別途契約すると月額数千円、日程調整ツールも月額1,000円前後。これらを個別に揃えるよりも、**営業ワークフロー全体を1つのツールでカバーできる**のは、管理の手間とコストの両面でメリットがあります。

## Focalizeの使い方3ステップ

実際の使い方はとてもシンプルです。

### ステップ1: メールを転送する

問い合わせや商談依頼のメールが届いたら、Focalizeの専用アドレスに転送します。Gmailの転送ボタンを押してアドレスを指定するだけです。自動転送ルールを設定しておけば、この手間すらなくなります。

### ステップ2: AIが情報を自動抽出する

転送されたメールをAIが解析し、以下の情報を自動で抽出します。

- **会社名**（または個人名）
- **担当者の連絡先**（メールアドレス、電話番号）
- **希望日時**（候補日が複数あれば全て）
- **案件の概要**

手入力のミスや入力漏れがなくなるのも大きなメリットです。

### ステップ3: カンバンボードで案件管理

抽出された情報は、自動的にカンバンボード上に「案件（ディール）」として登録されます。

カンバンボードでは、案件を「リード」「商談中」「提案済み」「受注」「失注」などのステージで視覚的に管理できます。ドラッグ&ドロップでステータスを更新するだけなので、Excelのように「どのセルを更新すればいいんだっけ？」と迷うことがありません。

さらに、Googleカレンダーとの連携により、商談の日程調整もFocalize上で完結します。

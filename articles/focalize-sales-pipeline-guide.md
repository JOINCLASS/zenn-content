## パイプライン管理とは何か

営業パイプラインとは、見込み客との最初の接点から成約に至るまでの流れを、段階（ステージ）に分けて管理する仕組みです。

一般的なステージ構成はこのようになります。

```
リード → 初回連絡 → 商談中 → 提案・見積 → 交渉 → 受注 / 失注
```

各ステージにいま何件の案件があるかを把握することで、「来月の売上見込みはどのくらいか」「どの案件にフォローが必要か」「営業活動のボトルネックはどこか」が見えるようになります。

これが「商談の見える化」の基本です。大企業ではSalesforceやHubSpotで当たり前に行われていますが、フリーランスや少人数チームではまだExcelやスプレッドシートで管理しているケースが多いのが実情です。

## Excel管理の限界

Excel（Googleスプレッドシート含む）で商談管理をしている方は多いと思います。最初のうちは問題なく回ります。しかし、案件が10件を超えたあたりから、以下の問題が表面化します。

**ステータスの更新を忘れる。** 商談が進んでもシートを更新するのを忘れ、実際の状況と管理表がずれる。週末にまとめて更新しようとして、どこまで進んでいたか思い出せない。

**全体像が見えない。** 行が増えるほどスクロールが必要になり、「いま提案中の案件が何件あるか」を把握するのにフィルタやピボットを駆使する必要がある。一目で分かる状態にはならない。

**手入力のコストが高い。** 新しい問い合わせが来るたびに、メールの内容を読み取って会社名・担当者名・連絡先をセルに手入力する。1件15分の作業でも、月に15件あれば約4時間です。

**チーム共有がしづらい。** 複数人で同じシートを編集すると、上書き事故や「誰がどこを更新したか分からない」問題が起きる。コメントやチャットで補足が必要になり、情報が分散する。

つまりExcelは「記録する場所」にはなっても、「営業活動を前に進めるツール」にはなりにくいのです。

## カンバンボードの威力

パイプライン管理で最も直感的なのが、カンバンボード形式です。

```
| リード | 商談中 | 提案済み | 受注 |
|--------|--------|----------|------|
| A社    | C社    | E社      | G社  |
| B社    | D社    |          |      |
|        |        |          |      |
```

縦にカード（案件）、横にステージ。この配置だけで、いま営業活動がどういう状態にあるかが一目で分かります。

カンバンの利点は3つあります。

**1. 視覚的に把握できる。** どのステージに案件が集中しているか、滞留しているカードはないか、が瞬時に分かる。「提案済みのカードが5枚溜まっている。先方の返答待ちが多い」といった気づきが自然に得られます。

**2. 操作が簡単。** 案件が次のステージに進んだら、カードをドラッグ&ドロップするだけ。Excelのセルを探して値を書き換える手間がありません。

**3. チームで共有しやすい。** 全員が同じボードを見ているので、「あの案件いまどうなってる？」という確認が不要になる。情報の非対称性が解消されます。

## 自動化で「入力ゼロ」を実現する

カンバンボードで管理すれば視認性は解決します。しかし、もう一つの問題が残っています。それは「最初の入力」です。

新しいリードが来るたびに、メールを読んで案件カードを手動で作成する。これが面倒で続かない、というのがパイプライン管理が定着しない最大の理由です。

ここでAIによる自動化が効きます。

たとえば、営業ツールの導入を検討して「まずはTrelloでカンバン管理してみよう」と始めても、新規リードのたびにカードを手動作成するのが手間で、1ヶ月後には使わなくなっている。こうした「ツール疲れ」を経験した方は少なくないはずです。

[Focalize](https://focalize.jp) では、見込み客からのメールを専用アドレスに転送するだけで、AIがメール本文から以下を自動抽出します。

- 会社名
- 担当者名
- 連絡先（メール・電話）
- 希望日時
- 案件の概要

抽出された情報はカンバンボード上に案件カードとして自動登録されます。手入力は一切不要です。

抽出精度も高く、「株式会社〇〇の田中です。来週火曜か水曜でお時間いただけますか」といった自然文から、会社名・担当者名・候補日をそれぞれ正確に拾ってくれます。

さらにGoogleカレンダーと連携しているため、日程調整もFocalize上で完結します。案件の登録から商談の日程確定までが、「メールを転送する」というワンアクションで始まるわけです。月10件まで無料、クレジットカード登録も不要なので、導入のハードルはほぼありません。

## まとめ：小さく始めて営業を仕組み化する

営業パイプライン管理は、大企業だけのものではありません。フリーランスや少人数チームこそ、限られた時間とリソースの中で商談の取りこぼしを防ぐために、パイプラインの可視化が重要です。

ステップとしては以下の順番がおすすめです。

1. まず自分の営業ステージを定義する（3〜5段階で十分）
2. Excelからカンバンボードに移行して視認性を確保する
3. AI自動化で入力コストをゼロに近づける

Focalizeは月10件まで無料で使えます。Googleアカウントでログインするだけで始められるので、まずは今ある案件をカンバンに並べてみてください。商談の全体像が見えるだけで、次に何をすべきかが明確になるはずです。

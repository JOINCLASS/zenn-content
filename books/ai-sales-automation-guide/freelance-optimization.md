## フリーランスの「営業あるある」

フリーランスや個人事業主として独立すると、本業のスキル以外に「営業」という仕事が丸ごとのしかかってきます。

会社員時代は営業部門がやってくれていたこと、全部自分でやらなければならない。

- 問い合わせメールが来たけど、返信が3日後になってしまった
- 「あの案件、いま見積もり出したっけ？」と記憶が曖昧になる
- Excelの案件管理表、先月から更新していない
- 名刺交換したあの人にフォローメールを送ろうと思って、もう2週間経っている

フリーランスにとって営業事務は「やらないと売上が止まる」のに「時間を取られすぎると本業ができない」という厄介な存在です。

本章では、この営業事務の内訳を具体的な数字で分析し、AIで自動化できる領域を明確にします。

## 営業事務の内訳を分解する

まず、フリーランスの営業事務にどれだけ時間を使っているかを分解してみましょう。

| 作業内容 | 月あたり目安 | 割合 |
|---------|------------|------|
| メールの確認・返信 | 4時間 | 25% |
| 案件情報の転記（メール→Excel/CRM） | 4時間 | 25% |
| 日程調整のやりとり | 3時間 | 19% |
| ステータス管理（進捗更新） | 2時間 | 13% |
| フォローアップの確認・送信 | 2時間 | 13% |
| 提案書・見積書の作成 | 1時間 | 5% |

合計で月に約16時間。週に4時間、つまり **週1日の半分が営業事務に消えている** 計算です。

この表をよく見てください。**転記・入力・ステータス更新で全体の約60%を占めています**。さらに日程調整まで含めれば約80%です。

## 80%は「判断が不要な作業」

営業事務の80%を占める作業の共通点は、**「判断」が必要ない**ことです。

- メールに書いてある会社名を、別のツールに書き写す → **転記**
- カレンダーの空きを確認して、候補日を提示する → **照合**
- 案件のステータスを「商談中」から「提案済み」に変更する → **更新**

これらはどれも、入力情報と出力が明確に定義できる **定型的な作業** です。定型的であるということは、AIに任せられるということです。

逆に、残りの20%を占める作業は人間がやるべき仕事です。

- 見込み客の課題をヒアリングする → **対話・理解**
- 提案内容を練り上げる → **創造・戦略**
- 価格交渉を行う → **判断・交渉**

この20%こそが営業の本質であり、売上に直結する仕事です。

## Before/After：自動化で何が変わるか

### Before: すべて手作業の場合

```
1. 問い合わせメールを受信（Gmail）
2. メールを読んで、会社名・担当者名・連絡先をExcelに手入力
3. 相手の希望日時を確認し、自分のカレンダーと照合
4. 候補日を返信メールで送る
5. 日程が決まったらカレンダーに手動登録
6. Excel上のステータスを「商談調整中」に更新
→ ここまでで約15〜20分。月10件なら3時間以上。
```

### After: AIに転記を任せた場合

```
1. 問い合わせメールを専用アドレスに転送（1クリック）
2. AIが会社名・担当者名・連絡先・希望日時を自動抽出
3. カンバンボードに案件として自動登録
4. Googleカレンダーと連携して日程候補を自動提示
→ 所要時間：転送の10秒だけ。月10件でも2分未満。
```

**15〜20分の作業が10秒になる**。これが「80%削減」の中身です。

## 月間の時間節約効果

自動化前後の時間配分を比較してみましょう。

| 作業内容 | Before | After | 削減率 |
|---------|--------|-------|-------|
| メールの確認・返信 | 4時間 | 2時間 | 50% |
| 案件情報の転記 | 4時間 | 0.1時間 | 97% |
| 日程調整のやりとり | 3時間 | 0.5時間 | 83% |
| ステータス管理 | 2時間 | 0.5時間 | 75% |
| フォローアップの確認・送信 | 2時間 | 0.5時間 | 75% |
| 提案書・見積書の作成 | 1時間 | 1時間 | 0% |
| **合計** | **16時間** | **4.6時間** | **71%** |

月に約11時間、年間で約130時間の節約です。

この130時間を「本当の営業」に充てたらどうなるでしょうか。

- 月10件だった商談を14件に増やせる
- 提案書のクオリティを上げて成約率を改善できる
- 既存顧客へのフォローに時間を使い、リピート率を高められる

営業事務の削減は、単なるコスト削減ではなく、**売上を伸ばすための時間を創出する行為**です。

## フリーランスの営業コスト試算

時間の節約を金銭的な価値に換算してみましょう。

フリーランスの時間単価を5,000円と仮定します（年収600万円÷年間稼働1,200時間）。

```
月11時間 × 5,000円 = 月55,000円分の時間価値
年間130時間 × 5,000円 = 年65万円分の時間価値
```

この時間を本業の案件に充てれば、そのまま売上に変わります。営業自動化ツールの月額費用が仮に3,000円だとしても、**投資対効果は18倍以上**です。

## よくある反論と回答

### 「まだ案件が少ないから手動で十分」

案件が少ないうちこそ、仕組みを作る絶好のタイミングです。案件が増えてから仕組みを作ろうとすると、日々の対応に追われて後回しになります。

### 「無料ツールで十分」

ExcelやNotion、Trelloなどの汎用ツールでも営業管理はできます。しかし「入力作業」は残ります。AI自動化ツールとの決定的な違いは、「入力コストがゼロになるかどうか」です。入力コストがゼロにならない限り、管理ツールが形骸化するリスクは消えません。

### 「AIの精度は信用できるのか」

メール本文からの情報抽出精度は非常に高いレベルに達しています。「株式会社〇〇の田中です。来週火曜か水曜でお時間いただけますか」といった自然文から、会社名・担当者名・候補日をそれぞれ正確に抽出できます。

万が一の誤抽出は、カンバンボード上で簡単に修正できます。ゼロから入力するのと、AIの抽出結果を確認・修正するのとでは、かかる時間が桁違いです。

## 段階的な導入アプローチ

いきなりすべてを自動化しようとする必要はありません。以下のステップで段階的に進めるのがおすすめです。

### ステップ1：現状の棚卸し（1日）

自分の営業事務にどれだけ時間を使っているかを1週間だけ記録してみてください。本章の分解表を参考に、各作業にかかった時間をメモするだけで十分です。「思っていたより時間を使っていた」という気づきが得られるはずです。

### ステップ2：ツール導入と最初の1件（1日）

AI営業自動化ツールにサインアップし、最初の1件だけメール転送を試してみます。Googleアカウントでログインするだけで始められ、クレジットカードの登録も不要です。

### ステップ3：1週間の試用（1週間）

新規の問い合わせや商談依頼が来たら、すべて転送してみます。1週間後にはカンバンボードに案件が並び、パイプラインの全体像が見えるようになっているはずです。

### ステップ4：ワークフローの定着（2〜4週間）

転送が習慣になったら、次はフォローアップメールのテンプレート化、日程調整の自動化へと進みます。第5章・第6章で詳しく解説します。

## まとめ

本章のポイントを整理します。

- フリーランスの営業事務は月16時間。その80%は「転記」「日程調整」「ステータス更新」の定型作業
- 定型作業はAIに任せることで、月11時間・年間130時間を創出できる
- 空いた時間を商談の質向上やリード開拓に充てれば、売上増に直結する
- 段階的に導入し、まずは1件のメール転送から始めるのが成功のコツ
